# Flight Delay Prediction Using Deep Learning

## Project Description
This project predicts whether a flight will be delayed using Deep Learning techniques.  
Historical flight data is analyzed to identify patterns that contribute to flight delays.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- TensorFlow / Keras
- Jupyter Notebook

## Dataset
The dataset contains historical flight information such as:
- Flight date
- Airline
- Origin and destination
- Scheduled and actual departure time
- Delay status

## Project Structure
Flight-Delay-Prediction/
- Flight_Delay_Prediction_using_Deep_Learning.ipynb
- dataset/
- README.md

## Workflow
1. Data loading and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature encoding and scaling
4. Train-test split
5. Deep Learning model training
6. Model evaluation

## Results
The model predicts flight delays with good accuracy using a Deep Learning approach.

## How to Run
1. Clone the repository
2. Install required libraries
3. Open the Jupyter Notebook
4. Run all cells
